Introducción

Funciones

Los programas extensos pueden ser complejos, lo que ocasiona que sean difíciles de comprender y depurar. La técnica más importante para reducir la complejidad es dividir un programa en secciones (relativamente) aisladas. Esto nos permite enfocarnos en una sección aislada sin las distracciones de todo el programa completo. Además, si la sección tiene nombre, podemos llamarla. En JavaScript estas secciones se conocen como funciones, podemos definir el concepto de función como un bloque de código, que recibe cero o más argumentos y que realiza una operación para devolvernos un valor y/o realizar una tarea específica. Las funciones deben tener un nombre asignado para poder “llamarlas” durante la ejecución del programa y son completamente reutilizables.

Sintaxis básica

Hasta ahora hemos visto algunas funciones predeterminadas, por ejemplo alert(“texto”), pero ahora vamos a crear nuestras propias funciones. La sintaxis básica para crear una función es la siguiente:
function [nombre_función]() {
         //código a ejecutar por la función
}
// En otro punto de la página llamamos la función
[nombre_función]();
