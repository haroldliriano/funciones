Argumentos de las Funciones

¿Qué son los argumentos?

Muchas veces queremos personalizar las funciones ejecutando todo el código, pero cambiando algunos detalles. De esa forma tendríamos un código que es reutilizable y flexible. Podemos lograr esto al especificar "parámetros" o argumentos para una función, luego usar estos argumentos para cambiar los valores de las variables dentro de la función y pasarlos cuando llamemos la función. Podemos definir los argumentos de las funciones como un conjunto de valores delimitados por comas que se emplean para pasar información a las funciones.

Ejemplo

Por ejemplo, ¿qué pasa si quisiéramos incluir un nombre en la función del código anterior, es decir, personalizar el saludo, Podríamos incluir el parámetro “nombre” y al llamarla ponerle un nombre como parámetro.

function saludar(nombre){
       document.write(“Hola!”);
}
//llamando la función
saludar(“Harold”); // la salida de la función seria “Hola Harold ”


Nuestras funciones pueden aceptar cualquier número de argumentos: cero, uno, dos o más. La siguiente función calcula la suma de dos números:

function sumar(x, y){
         document.write("La suma de "+ x +"y"+ y +" es igual a "+ (x+y) );

}

sumar(5,6);

//la salida de esta función es “La suma de 5 y 6 es igual a 11”

Consideraciones sobre los argumentos

Realmente depende de lo que quieras que hagan tus funciones y qué tanto deseas personalizar lo que pueden hacer. Siempre puedes empezar sin argumentos, y luego agregar más conforme te des cuenta que los necesitas.
